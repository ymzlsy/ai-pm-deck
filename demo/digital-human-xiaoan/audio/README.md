# 预录拟人音频占位目录

Demo 默认用浏览器原生 TTS（机械感重）。把豆包/即梦合成的 mp3 放进本目录后，会**自动优先播放**，演示拟人度。

## 使用步骤

1. 到 [豆包语音合成](https://www.volcengine.com/product/voice-synthesis) 或 [即梦](https://jimeng.jianying.com/) 等平台
2. 选女声音色（推荐豆包「灿灿」/「通用 Bella」/「知性女声」）
3. 语速 1.05~1.10×，pitch +5%
4. 把 5 段示例（见下表）逐个合成，命名为对应文件名后放到本目录
5. 编辑 `../index.html` 中的 `PRERECORDED_AUDIO` 对象，取消对应行注释

## 启用方式（合成完毕后）

编辑 `../index.html` 第 818 行附近 `PRERECORDED_AUDIO` 常量，取消相关行的 `//` 注释：

```js
const PRERECORDED_AUDIO = {
  'brake': 'audio/brake.mp3',
  'signal': 'audio/signal.mp3',
  'overspeed': 'audio/overspeed.mp3',
  'emergency': 'audio/emergency.mp3',
  'weather': 'audio/weather.mp3',
};
```

并在场景定义中增加 `audioKey: 'brake'` 等字段（已预留好接口）。

## 文件大小建议

- 单段 30~60 秒，mp3 128kbps 即可
- 单文件 < 1 MB，整个 audio/ 目录 < 5 MB
